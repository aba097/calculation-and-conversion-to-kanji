# 演算した結果の数値を漢字表記に変換
## サイト説明
- 計算式を入力し、計算結果を漢字表記に変換する
    - <数値><演算子><数値><演算子>のように計算を続けて行うことができる
    - 計算式でなくても数値1つのみでもよい
- 対応
    - 四則演算(+, -, *, x, /)に対応
    - 少数に対応
    - 負の数に対応
    - 単位は兆まで対応
## プログラム
- server.goは開発環境用のサーバ
    - `go run .\server.go`で起動
    - http://localhost:8000/ へアクセス
